# Protocole 12

Carnet d'entraînement web — programme kettlebell Hardstyle 12 semaines adapté Takayasu.

## Utilisation

Ouvre `index.html` dans n'importe quel navigateur, ou accède à la version en ligne sur GitHub Pages.

## Fonctions

- Programme complet 12 semaines (3 blocs × 4 semaines)
- Suivi des séances cochées + streak hebdo
- Journal RPE / charges / notes
- Timer intégré : Repos, EMOM, AMRAP
- Tests bras gauche hebdo avec graphique de progression
- Données stockées en local (localStorage), aucune connexion, aucun compte
- Export / import JSON pour sauvegarde manuelle

## Stack

HTML + CSS + JavaScript vanilla. Aucune dépendance. Un seul fichier.

## Vie privée

Toutes les données restent sur ton appareil. Rien n'est envoyé sur un serveur.
